# slm-benchmarks
A public repository on Benchmarking all kinds of SLM Models

## Documentation

- **[Popular LLM Models](POPULAR_LLM_MODELS.md)** - Comprehensive list of the most popular Large Language Models (LLMs) available today, including proprietary, open-source, and small language models.

## About

This repository provides benchmarking resources and information for Small Language Models (SLMs) and Large Language Models (LLMs). Our goal is to help developers and researchers make informed decisions about which models to use for their specific use cases.
